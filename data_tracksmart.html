Project Budget Tracking Tool

A project management tool for large AEC company to keep track of the budget, time and resources.

Overview

Large AEC Built Environment industries are tasked with projects that often involves multiple parties, company and contractors. For projects that are ongoing for several years, it can be a challenge and lose track to keep up with the budget and resources as people come and go, their salary increases or a change in project scope. This tool will allow project director, owner and manager to track every step within the project with a better user experience and clean UI. Currently, tracking tool is only used in Excel Spreadsheet.

Note: This project was part an internal project at Arup and all information displayed has been amended for confidential purposes.


Objective

To create a better user experience and clean design for large project data from excel so that project manager, director and owner can use to keep track of the project.


Team

Business Analyst, UX/UI Designer, 2x Developers


Design

This is the landing page after loggin in to see the list of table for all the projects given. Project views allow you to see the overall details of the project whereas, the commercial view goes more into fee and pricing details.



Job Code Mapping with Task ID page allows the user to map the job code to each specific resources. Job code (or Task ID) will define each individual task. So the resources can bill the hours to that specific job code. Task is divided into two cateogies; labour item and expense item, using color coding. The user can add the Task ID and edit the description within the Task ID list.



This page is the final step within the project management tracking progress. If you have resources with no Task ID, then a flag would appear on the top-right corner as a warning. This is where the user will mostly spend their time editing and configuring to assign each hour mathematically.
